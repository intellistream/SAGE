## 🚀 使用指南

### 1. 创建 Flow

在 SAGE Studio 前端:
1. 拖拽节点到画布
2. 连接节点
3. 配置节点属性
4. 保存 Flow

### 2. 在 Playground 测试

1. 点击工具栏 **"💬 Playground"** 按钮
2. 输入测试消息
3. 点击发送
4. 查看真实的 Flow 执行结果
5. 展开 Agent 步骤查看节点执行详情

### 3. 查看执行步骤

每个 Agent 步骤显示:
- ✅ 节点名称
- ✅ 输入参数 (JSON)
- ✅ 输出结果 (JSON)
- ✅ 执行时间 (毫秒)
- ✅ 执行状态
