## 实际发现的API

### JobManager 实际方法:
- `submit_job()` - 提交作业
- `pause_job()` - 暂停作业
- `continue_job()` - 继续作业
- `delete_job()` - 删除作业
- `get_job_status()` - 获取作业状态
- `list_jobs()` - 列出所有作业
- `cleanup_all_jobs()` - 清理所有作业

### JobInfo 实际构造函数:
```python
JobInfo(environment, graph, dispatcher, uuid)
```

### ExecutionGraph 特点:
- 需要 `BaseEnvironment` 参数
- 自动设置日志系统
- 包含 `logger` 属性
