- **Installation Automation (`install/`)**: A modular Bash framework organized into `lib/` (shared utilities like `common_utils.sh`, `logging.sh`), `installers/` (core logic), `checks/` (dependency verification), and `fixes/` (environment repair scripts). It supports Conda/Pip workflows and includes a monitoring script for real-time progress tracking.
- **Maintenance & Git Hooks (`maintenance/`)**: Centralized via `sage-maintenance.sh`, which acts as a CLI entry point for cleaning, security checks, and health diagnostics. It integrates with Git hooks (`pre-commit`, `post-checkout`) to enforce repository hygiene and uses helper scripts for common tasks like config security auditing.
- **Benchmark Carrier (`benchmark_carrier/`)**: A Python-based synthetic workload generator (`main.py`) that validates the VAMOS launcher contract by replaying JSON-based run plans and producing trace/metric artifacts without requiring live execution.
- **Compliance Scripts (`scripts/`)**: Standalone Python and Bash utilities for architectural governance, such as validating PEP 420 namespace compliance and checking cross-repo dependency synchronization.
