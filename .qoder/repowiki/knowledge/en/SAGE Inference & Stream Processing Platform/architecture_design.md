- **Unified Runtime Core**: The `runtime` module (FlowNet) acts as the central orchestration layer, exposing a consistent API for local and distributed execution. It manages actor lifecycles, shared state services, and backend container affinity.
- **Contract-Based Integration**: External inference engines (e.g., vLLM, isagellm) and workflow products (ComfyUI, LangGraph) are integrated via strict contracts defined in the `serving` module, ensuring isolation while enabling seamless data flow into the runtime.
- **Declarative Stream Processing**: The `stream` module provides a DAG-based API for data transformation, which the runtime compiles into executable actor graphs, bridging high-level logic with low-level execution primitives from `foundation`.
- **Centralized Tooling & Configuration**: A single `sage` CLI and `sage-dev` developer tool (defined in `cli` and `tools`) manage the entire lifecycle, from environment setup and dependency verification to code quality gates and benchmark execution, enforcing uniform conventions across all modules.
