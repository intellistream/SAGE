# 基础示例 (Basic Examples)

这个目录包含SAGE框架的基础入门示例，适合初学者了解框架的基本概念和使用方法。

## 📁 文件列表

### `hello_world.py`
最基础的Hello World示例，展示：
- 如何创建简单的BatchFunction
- 如何使用MapFunction进行数据转换
- 如何使用SinkFunction输出结果
- 本地环境的基本使用

#### 运行方式：
```bash
python hello_world.py
```

#### 学习目标：
- 理解SAGE的基本组件：Source、Map、Sink
- 掌握本地环境的创建和使用
- 了解数据处理管道的基本结构

## 🔗 相关配置

基础示例通常不需要复杂配置，直接运行即可。
