## 项目概述## 项目概述



本项目旨在基于 SAGE 框架，构建一个**独立的高性能 LLM 推理执行引擎（sageInfer）**，与 vLLM 对标竞争，将国产硬件（华为昇腾、寒武纪、海光、昆仑芯等）作为一等公民进行深度优化。本项目旨在基于 SAGE 框架，构建面向大规模国产算力的高性能推理引擎，实现国产硬件（华为昇腾、寒武纪、海光、昆仑芯等）上的高效 LLM 推理能力。
